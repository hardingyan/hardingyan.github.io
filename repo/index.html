我的开源项目

这里列出了我正在开发和维护的主要开源项目，欢迎查看和贡献代码。

XNN 框架

XNN 是一个轻量级的神经网络框架，旨在提供高效、灵活的深度学习工具。

  • 代码库：github.com/hardingyan/XNN
  • 特点：高性能、低内存占用、适合边缘设备部署
  • 语言：C++, Python

XCC 编译器

XCC 是一个实验性的编译器集合，支持多种编程语言的优化编译。

  • 代码库：github.com/hardingyan/XCC
  • 特点：模块化设计、高效优化、多语言支持
  • 语言：C++, Rust

XDB 数据库

XDB 是一个分布式数据库系统，注重高可用性和水平扩展能力。

  • 代码库：github.com/hardingyan/XDB
  • 特点：高并发、强一致性、易于扩展
  • 语言：Go, Rust

XOS 操作系统

XOS 是一个微内核实验性操作系统，专注于安全性和模块化设计。

  • 代码库：github.com/hardingyan/XOS
  • 特点：微内核架构、安全隔离、可扩展
  • 语言：C, Rust, Assembly
